#+title: Theme Preview Workflow

#+DATE: 2026-07-16
#+AUTHOR: Guangtao Tao
#+FILETAGS: :org-zhixing:operations:themes:

* List available themes

#+begin_src sh
direnv exec . just theme-list
#+end_src

* Start a temporary preview

Elegant Blog:

#+begin_src sh
direnv exec . just theme-preview elegant-blog 5173
#+end_src

Minimal Notes:

#+begin_src sh
direnv exec . just theme-preview minimal-notes 5173
#+end_src

Use the in-page Appearance navigator for Paper or Midnight.

The documentation theme is selected independently of its color flavor:

#+begin_src sh
direnv exec . just theme-preview documents 5173
#+end_src

Use the in-page Appearance navigator for Latte, Frappé, Macchiato, or Mocha.

The command prints the exact URL to open. It preserves the tracked
=public/org-zhixing.toml=, uses a temporary config while the server is running,
and restores the canonical generated registry when the process exits or receives
=Ctrl-C=. Theme preview uses the root development base path; the production
=base_path= remains unchanged.

* Validate a theme package

#+begin_src sh
direnv exec . just theme-check themes/minimal-notes
#+end_src
